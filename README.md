**XHeader** হলো Termux-এর জন্য একটি প্রিমিয়াম ব্যানার মেকার টুল। এটি আপনার সাধারণ Termux টার্মিনালকে একটি স্টাইলিশ এবং প্রফেশনাল লুক দেয়। এতে রয়েছে চমৎকার অ্যানিমেটেড ওয়েলকাম UI, পাসওয়ার্ড প্রোটেকশন এবং ওহ-মাই-জেড-এসএইচ (Oh My Zsh) কনফিগারেশন।

---

## 📥 ইনস্টলেশন কমান্ড (Installation Commands)

আপনি চাইলে এক লাইনে অথবা আলাদা আলাদাভাবে কমান্ডগুলো ব্যবহার করে টুলটি ইনস্টল করতে পারেন।

### ১. এক লাইনে ইনস্টল করার কমান্ড (Single Line Command)
দ্রুত ইনস্টল করতে নিচের পুরো কমান্ডটি কপি করে Termux-এ পেস্ট করুন:
```bash
apt update && apt upgrade -y && pkg install git -y && git clone https://github.com/xhacknet/XHeader && cd XHeader && chmod +x Banner.sh && ./Banner.sh 
```

### ২. আলাদা আলাদা কমান্ড (Step by Step)

যদি উপরের কমান্ডে সমস্যা হয়, তবে নিচের ধাপগুলো অনুসরণ করুন:
```bash
apt update && apt upgrade -y
```
```bash
pkg install git -y
```
```bash
git clone https://github.com/xhacknet/XHeader
```
```bash
cd XHeader
```
```bash
chmod +x Banner.sh
```
```bash
./Banner.sh
```
